
## 从一个区域到另一个区域的通路
|区域     |位置  |坐标|
|:----   |:----  |:----|
|left    |左侧   |x <= 13|
|right   |右侧    |x >= 87|
|top     |上侧    |13 < x < 87 && y < 50 |
|bottom  |下侧    |13 < x < 87 && y >= 50|

|从bottom	|左下		| 右下		| 目标地		|目标地|
